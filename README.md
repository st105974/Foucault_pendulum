# Foucault_pendulum

Коды для численного моделирования маятника Фуко с параметрической подкачкой.

## Файлы

### `compare_energy_decay.py`
Сравнение двух механизмов диссипации энергии маятника: линейного вязкого трения и квадратичного аэродинамического сопротивления. Моделирует свободные затухающие колебания при каждом из механизмов и строит график полной энергии E(t) на временном интервале 300 с. Результат сохраняется в `compare_energy_decay.png`.

### `plot_solution.py`
Численное интегрирование нелинейного уравнения маятника с переменной длиной подвеса (параметрическая подкачка) в безразмерном времени. Строит временно́й ряд отклонения x(t) при заданных значениях ε (амплитуда модуляции) и δ (расстройка частоты). Результат сохраняется в `solution_eps005_delta0.png`.

### `find_limit_cycle.py`
Поиск устойчивого предельного цикла параметрически возбуждаемого маятника. Для каждого значения расстройки δ из заданной сетки проверяет три критерия: периодичность отображения Пуанкаре, стабильность амплитуды и близость к целевой амплитуде 1 м. Строит карту «принятых» значений δ и ошибку периодичности в зависимости от δ. Результат сохраняется в `limit_cycle_search.png`.

### `spherical_pendulum.py`
Численное интегрирование сферического маятника переменной длины с учётом параметрической подкачки, линейного трения и кориолисовых эффектов вращения Земли (широта 60°). Строит три графика эволюции траектории в плоскости Oyz на временных интервалах 0–300 с, 300–1000 с и 1000–7200 с, а также сравнение мгновенной скорости прецессии плоскости качания с аналитическими вкладами эффекта Фуко и прецессии Эйри. Также создаёт анимацию эволюции траектории (GIF и MP4). Результаты сохраняются в `yz_trajectory_0_300.png`, `yz_trajectory_300_1000.png`, `yz_trajectory_1000_7200.png`, `precession_contributions.png`, `trajectory_evolution.gif`, `trajectory_evolution.mp4`.
