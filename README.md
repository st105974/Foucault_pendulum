# Foucault_pendulum

Коды для численного моделирования маятника Фуко с параметрической подкачкой.

## Файлы

### `compare_energy_decay.py`
Сравнение двух вариантов диссипации энергии маятника: только линейного сопротивления или только квадратичного сопротивления. Моделирует свободные затухающие колебания для каждого случая и строит график полной энергии E(t) на временном интервале 300 с.

### `plot_solution.py`
Численное интегрирование нелинейного уравнения маятника с переменной длиной подвеса (параметрическая подкачка) в безразмерном времени. Строит временно́й ряд отклонения x(t) при заданных значениях ε (амплитуда модуляции).

### `find_limit_cycle.py`
Поиск устойчивого предельного цикла параметрически возбуждаемого маятника. Для каждого значения расстройки δ из заданной сетки проверяет три критерия: периодичность отображения Пуанкаре, стабильность амплитуды и близость к целевой амплитуде 1 м. Строит карту «принятых» значений δ и ошибку периодичности в зависимости от δ.

### `spherical_pendulum.py`
Численное интегрирование сферического маятника переменной длины с учётом параметрической подкачки, линейного трения и кориолисовой силы инерции (широта 60°). Строит три графика эволюции траектории в плоскости Oyz на временных интервалах 0–300 с, 300–1000 с и 1000–7200 с, а также зависимость скорости прецессии плоскости качания от времени.
