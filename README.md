# Foucault_pendulum

Коды для численного моделирования маятника Фуко с параметрической подкачкой.

## Файлы

### `compare_energy_decay.py`
Сравнение двух вариантов диссипации энергии маятника: только линейного и только квадратичного сопротивлений. Моделирует свободные затухающие колебания для каждого варианта диссипации строит график полной энергии E(t) на временном интервале 300 с.

### `plot_solution.py`
Численное интегрирование нелинейного уравнения маятника с переменной длиной подвеса (параметрическая подкачка энергии) в безразмерном времени. Строит график колебаний x(t) при заданных значениях относительной модуляции длины ε = 0,0033 и δ = 0.

### `find_limit_cycle.py`
Поиск устойчивого предельного цикла параметрически возбуждаемого маятника. Для каждого значения расстройки δ из заданной сетки проверяет три условия: периодичность отображения Пуанкаре, стабильность амплитуды колебаний и близость амплитуды к целевому значению равному 1 м. Строит карту «принятых» значений δ и ошибку периодичности в зависимости от δ.

### `spherical_pendulum.py`
Численное интегрирование сферического маятника с учётом параметрической подкачки энергии, линейного сопротивления  и кориолисовой силы инерции. Строит графики эволюции траектории в плоскости Oyz на временных интервалах 0–300 с, 300–1000 с и 1000–7200 с, а также график скорости прецессии плоскости качания в сравнении с аналитическими вкладами эффекта Фуко и эллиптической прецессией.

### `Fu_16min.mp4`
Видеодемонстарция работы маятника.
